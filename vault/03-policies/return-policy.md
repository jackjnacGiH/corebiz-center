# นโยบายการคืนสินค้าและคืนเงิน

## ระยะเวลาคืนสินค้า

ลูกค้าสามารถคืนสินค้าได้ภายใน **7 วัน** นับจากวันที่ได้รับสินค้า

## เงื่อนไขการคืนสินค้า

1. สินค้าต้องอยู่ในสภาพสมบูรณ์ ไม่ผ่านการใช้งาน
2. มีบรรจุภัณฑ์เดิมและอุปกรณ์ครบถ้วน
3. มีใบเสร็จหรือหมายเลขคำสั่งซื้อ
4. **ไม่รับคืน** สินค้าประเภทเครื่องมือลม / สินค้าสั่งทำพิเศษ

## ขั้นตอนการคืน

1. ติดต่อเจ้าหน้าที่ผ่าน LINE / Messenger / โทร 02-xxx-xxxx
2. แจ้งหมายเลขคำสั่งซื้อและสาเหตุการคืน
3. ส่งสินค้ากลับตามที่อยู่ที่เจ้าหน้าที่แจ้ง
4. ทีมเราจะตรวจสอบสภาพสินค้าภายใน 3 วันทำการ
5. คืนเงินภายใน 7 วันทำการหลังตรวจสอบสำเร็จ

## ค่าใช้จ่ายในการส่งคืน

- กรณีสินค้าผิดพลาดจากเรา → **เรารับผิดชอบค่าส่ง**
- กรณีลูกค้าเปลี่ยนใจ → **ลูกค้ารับผิดชอบค่าส่ง**
